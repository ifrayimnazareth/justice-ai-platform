Algoritma Cipher Default

Metodologi pengacak data yang diterapkan seketika pada file KTP, Kartu Keluarga, dan Alat Bukti klien saat diunggah. AES-256 GCM membutuhkan CPU lebih tinggi namun tidak bisa di-brute force oleh Komputer Kuantum saat ini.

Purge Database Simulasi Kasus (Self-Destruct)

Menghapus seluruh histori MongoDB, mengosongkan direktori `/storage` termasuk Brankas AES, dan mengatur sistem kembali ke titik awal. Tindakan ini tidak bisa dibatalkan tanpa backup manual!
